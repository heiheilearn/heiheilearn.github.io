1. Docker概述

1.1 为什么学习Docker

  • 避免环境不同导致问题

  • 降低微服务阶段的学习成本

  • 为理解实际开发的打包发布流程打基础

1.2 Docker是什么

Docker是一个应用容器引擎

一个容器可以理解成一个轻量级虚拟机

1.3 Docker官网

https://docs.docker.com/

1.4 Docker核心概念

  • 镜像 相当于容器的模板
  • 容器 可以理解成轻量级虚拟机
  • 镜像仓库 存放镜像的仓库 https://hub.docker.com/

3.2.1 Docker操作常规流程

搜索镜像

  • 在网站上搜索镜像 http://hub.docker.com

  • docker search 镜像名

docker pull [镜像仓库/]镜像名[:标签]

docker images

docker iamges -q // 镜像id

docker ps

docker run -d nginx:latest

docker run -it ngins:latest

docker rm [选项] [容器名称/容器ID...]

docker exec [选项] 容器名 / 容器ID 命令

docker exeu 容器id ls
docker exeu -it 容器id bash

docker logs 容器id

docker logs -f 容器id

docker logs -n 20 容器id

docker stop 容器id

docker start
